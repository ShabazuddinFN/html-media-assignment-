1. HTML

    Primary Use Cases Key Features/Benefits
    Building the Structure of web pages
    • Simple and easy to learn
    • Compatible with all web browsers
    • Allows for semantic markup

  2. CSS

    Primary Use Cases Key Features/Benefits
    CSS is used to style and format the layout of web pages
    • CSS separates the content (HTML) from its presentation
    • CSS offers a wide range of styling options
    • CSS supports animations and transitions

  3. JavaScript

    Primary Use Cases Key Features/Benefits
    JavaScript enables interactive, dynamic content and functionality in web applications
    • JavaScript can perform client-side form validation before submitting data to the server, reducing server load and improving user experience.
    • JavaScript can be used across various platforms, including browsers, servers (Node.js)
    • JavaScript, often combined with HTML5 and CSS, enables complex animations and graphic effects, including games and interactive visualizations
