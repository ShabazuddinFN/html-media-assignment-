List of Tag :-

  1. h1 (Is a heading Tag level 1)
  2. p (Is a paragraph Tag)
  3. br (Is break the line )
  4. ul (Is a underline Tag)
  5. del (Its a identify the text will deleted )
  6. I (Its change the text in to italic sytle )
  7. B (Its bold the text)
  8. Small(Its reduce the text size)
  9. ul (Its defines an unordered list)
  10. mark (Its highlight the text)
  11. sup (Its defines superscript text. Superscript text appears half a character 'above' the normal line)
  12. Sub (Its defines subscript text. Subscript text appears half a character 'below' the normal line)
